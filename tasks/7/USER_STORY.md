# User Story: Login Screen

## Description

As a user, I want to access the application through a login screen so that I can authenticate and access protected features.

## Story Details

- **Issue:** #7
- **Title:** Criar uma Tela de Login
- **Category:** Authentication / UI
- **Priority:** High
- **Complexity:** Medium

## Background

The application requires a login mechanism to allow users to authenticate before accessing the main features. This is the entry point for user authentication.
